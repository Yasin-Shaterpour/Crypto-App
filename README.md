# Crypto-App
Modern crypto dashboard with real-time market data, search, and glassmorphism UI using React and CoinGecko API


---

## 🚀 Features

- 🔍 Real-time coin search (CoinGecko API)
- 💱 Currency switcher (USD / EUR / JPY)
- 📊 Live market data (price, market cap, volume)
- 🎨 Modern Glassmorphism UI
- ⚡ Fast and responsive design
- 🧠 Debounced API requests (performance optimized)
- ❌ AbortController for request cancellation
- 📱 Fully responsive layout
- ✨ Smooth hover animations & micro-interactions

---

## 🛠️ Tech Stack

- React.js (Hooks: useState, useEffect)
- CSS Modules
- CoinGecko API
- Vite
- JavaScript (ES6+)
